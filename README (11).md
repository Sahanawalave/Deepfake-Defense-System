## рҹҡҖ What is VIGIL-EYE?

**VIGIL-EYE** is a production-ready, AI-powered deepfake detection and liveness verification system. It deploys a **5-layer biological + algorithmic pipeline** that makes synthetic identity fraud computationally impossible вҖ” in real time, at sub-second speed.

> Not just a deepfake detector. A **zero-trust, multi-modal AI trust engine.**

---

## рҹ§  Problem Statement

Deepfake technology has crossed a critical threshold вҖ” AI-generated faces, cloned voices, and synthetic video are now **indistinguishable to the human eye**. This creates catastrophic risks:

```
рҹ“Ҳ  Deepfake fraud losses projected to exceed $40 Billion globally by 2027
рҹҸҰ  Banking KYC: Synthetic identity bypasses costing institutions billions/year
вҡ–пёҸ  Legal: AI-generated evidence actively disrupting court proceedings
рҹ—іпёҸ  Politics: Synthetic video deployed in election disinformation campaigns
рҹ’”  Society: Non-consensual deepfakes destroying personal reputations
рҹҺ“  Education: AI face-swapping used to cheat identity-verified proctored exams
```

Existing solutions rely on **single-model detection** вҖ” one adversarial attack defeats them entirely. There is no production solution combining biological liveness, behavioral analysis, and multi-model ML fusion into a real-time system. **Until now.**

---

## рҹ’Ў Solution

VIGIL-EYE deploys a **5-layer verification pipeline** attacking the deepfake problem from every dimension simultaneously:

| Layer | Component | What It Checks |
|:---:|:---|:---|
| 1пёҸвғЈ | **Input Integrity Guard** | Replay attack detection & stream injection prevention |
| 2пёҸвғЈ | **Biological Liveness** | rPPG pulse + eye blink + behavioral profiling |
| 3пёҸвғЈ | **Environmental Verification** | Ambient light, MoirГ© artifacts, scene coherence |
| 4пёҸвғЈ | **Multi-Model ML Engine** | Random Forest + Autoencoder + LOF + K-Means consensus |
| 5пёҸвғЈ | **Audio Integrity** | Spectral analysis, micro-tremor detection, acoustic fingerprinting |

Every layer must pass. A single failure triggers the **Auto Kill-Switch.**

## рҹӨ– AI Model Explanation

| Model | Role | Why It's Used |
|:---|:---|:---|
| рҹҢІ **Random Forest** | Texture classification across video frames | Ensemble decision trees resist adversarial feature manipulation |
| рҹ”„ **Autoencoder** | Anomaly detection вҖ” reconstructs "real" and flags deviation | Learns the distribution of authentic faces; deepfakes produce high reconstruction error |
| рҹ“Қ **Local Outlier Factor** | Detects behavioral + spatial outliers in real-time | Flags frames that are statistically inconsistent with a real person's motion patterns |
| рҹ”ө **K-Means Clustering** | Groups detected biometric profiles | Separates real identity clusters from synthetic/synthetic-adjacent outliers |

All 4 models vote simultaneously. The **weighted ensemble score** drives the master verdict. No single model can be fooled without defeating all four.

## вҡҷпёҸ Installation & Setup

### Prerequisites

| Tool | Min Version | Download |
|:---|:---:|:---|
| Python | 3.9+ | [python.org](https://python.org) |
| pip | 22+ | Bundled with Python |
| Git | Latest | [git-scm.com](https://git-scm.com) |
| Webcam | Any | Required for live sessions |

### Step 1 вҖ” Clone the Repository

```bash
git clone https://github.com/your-username/vigil-eye.git
cd vigil-eye
```

### Step 2 вҖ” Create Virtual Environment

```bash
python -m venv venv

# Windows
venv\Scripts\activate

# macOS / Linux
source venv/bin/activate
```

### Step 3 вҖ” Install Dependencies

```bash
pip install -r requirements.txt
```

### Step 4 вҖ” Configure Environment

```bash
cp .env.example .env
```

Edit `.env`:

```env
FLASK_ENV=development
FLASK_PORT=5000
MODEL_PATH=models/
DATASET_PATH=data/dataset.csv
SECRET_KEY=your_secret_key_here
```

### Step 5 вҖ” Launch

```bash
python app.py
```

Open **`http://localhost:5000`** in your browser рҹҡҖ
